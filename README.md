## Instructions to build

### 1 Install
```bash
npm install
```

### 2 Set MongoDB connection
```bash
export MONGO_URL="YOUR_MONGO_CONNECTION_STRING"
export DB_NAME="project2_lost_and_found"
```

### 3 Start server
```bash
node server.js
```

### 4 Open pages in browser
http://localhost:3000/login.html
